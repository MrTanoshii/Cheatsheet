## Cronjobs

```bash
# List all cronjobs
wp cron event list

# Run a specific cronjob immediately
wp cron event run <HOOK_NAME>

# Run all due cronjobs immediately
wp cron event run --due-now

# Remove a specific cronjob
wp cron event delete <HOOK_NAME>
```
